Alternative to Redis add-ons: Use Redis Labs

If both the add-ons, that is, Heroku Redis and Redis Enterprise Cloud don't work (as in requires credit card details), you can create an account Redis Labs. Walk through the basic free tier subscription process and spin up a redis database on any Cloud Provider that you like. Now, that you have landed on the dashboard, goto the Databases tab on the side pane and navigate to the newly spun database. On this page, obtain the Public Endpoint and Default User Password and create the CELERY_BROKER_URL as per the format:
redis://:<Default User Password>@<p Endpoint or IP Address(including PORT number)>/0
Add this as a config variable on the Settings/Config Vars tab under your app on Heroku.
